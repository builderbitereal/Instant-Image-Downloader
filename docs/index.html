BuilderBite

Microsoft Edge image downloader extension

Download visible web images, public Facebook gallery photos as ordered ZIP files, and image slides with left-arrow navigation.

Visit BuilderBite View README

Features

Gallery ZIP Download

Save visible batch images in order as numbered files inside one ZIP archive.

Facebook Public Post Support

Works with public and accessible post images that are already visible in your browser.

Manual Image Picker

Click a specific image or background image on the page to download exactly what you need.

Left Arrow Automation

Download the current image, move to the previous image, and repeat with an optional limit.

Built by BuilderBite. This tool does not bypass private content, authentication, or platform restrictions.
