# Security Policy

## Supported Versions

The latest public release is supported.

## Reporting a Vulnerability

Please report security issues privately through BuilderBite:

- Website: https://builderbite.com/

Do not post sensitive security details in public GitHub issues.

## Data and Privacy Notes

- The extension works in the browser and uses local extension storage for settings.
- It downloads images that are already visible and accessible in the active browser session.
- It does not bypass private content, authentication, site permissions, or platform restrictions.
